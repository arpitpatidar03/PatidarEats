# PatidarEats – Food Ordering Web Application

PatidarEats is a simple static food ordering web application built using React.js.
This project focuses on creating a modern and responsive frontend for a food delivery platform.

## Features

* Home page with food items
* Food menu displayed using card layout
* Add to Cart button (static functionality)
* Responsive user interface
* Clean and simple design

## Technologies Used

* React.js
* JavaScript
* HTML5
* CSS3

## How to Run the Project

1. Clone or download the project
2. Open the project folder
3. Install dependencies

npm install

4. Start the development server

npm start

The application will run at:
http://localhost:3000

## Project Description

PatidarEats is designed as a frontend-only food ordering interface where users can browse food items and view them in a visually appealing layout. This project is useful for learning React component structure and frontend design for food delivery platforms.
