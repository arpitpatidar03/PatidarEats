# PatidarEats 🍔

PatidarEats is a modern, full-stack E-Commerce food delivery web application built using the **MERN Stack** (MongoDB, Express.js, React.js, Node.js). [cite: 2026-03-07]

## 🚀 Features
* **Modern UI:** Clean and responsive design using React. 
* **Category Exploration:** Smooth animations and hover effects for food categories.
* **Cart Management:** Add items to cart, remove items, and see a professional bill breakdown.
* **Order History:** A dedicated "My Orders" section to track previous successful orders.
* **English Interface:** Professional English alerts and navigation.

## 🛠️ Tech Stack
* **Frontend:** React.js, React Router, Context API.
* **Styling:** Custom CSS with modern animations.
* **Data:** Fetching and displaying dynamic food data.
